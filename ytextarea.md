# 文本输入框组件

Ytextarea这个组件用来完成文本输入框及其样式，提示剩余输入字符及最多字数限制

## Example

使用
 
```html
<textarea
    class="weui_textarea"
    widget="ytextarea"
    name="textarea"
    placeholder="请填写内容"
    rows="3"
    maxlength="200"
>
</textarea>
```

##Options

|name|type|required|default|description|
|:---|:---|:---:|:---||
|placeholder|string|no|''|文本输入域的默认value值|
|maxlength|number|no|-1|输入字符的数的最大值|
|rows|number|no|''|文本输入框的可见行数(高度)|
